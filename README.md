# 📄 Product Requirements Document: Hello 프로젝트

## 🧭 프로젝트 개요
간단한 Python 기반 CLI 프로그램으로, 사용자에게 인사 메시지를 출력하는 기본 기능을 갖춘 학습용 프로젝트입니다.

## 🎯 목표
- Python 프로젝트의 기본 구조 학습
- VS Code와 연동한 디버깅 및 실행 실습
- 가상환경과 의존성 관리 실습

## 🗂️ 주요 기능
- "Hello, World!" 출력 기능
- 명령줄 인자 기반 메시지 출력
- 향후 사용자 입력 기반 응답 확장 고려

## 🔧 기술 스택
- Python 3.x
- venv (가상환경)
- VS Code (Python Extension)
- pytest (예정된 테스트 도구)
